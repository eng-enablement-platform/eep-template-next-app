## Getting Started

```bash
pnpm install
```

First, run the development server:

```bash
pnpm dev
```

```bash
# Terminal 1 — leave running
pnpm dev

# Terminal 2 — iterate freely against the warm server
pnpm e2e
pnpm e2e:ui
```
